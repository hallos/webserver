# webserver
Simple crossplatfom console-based webserver written in C++

Compiled with 
* MinGW64 GCC 4.7.3 on Windows
* GCC 7.2.0 on Ubuntu

## Build
The project is built with CMake

### Dependencies
* spdlog https://github.com/gabime/spdlog
